AD-049High Password Spray Attack Risk
🟠 HIGHPassword Policy▼
📋 Description

No account lockout AND weak password policy.
Domain is highly susceptible to automated password spray attacks.

⚡ Exploitable By
⚡ External attacker (no creds)
📍 Found In

LDAP → domainDNS → lockoutThreshold=0 + minPwdLength<8 (combined risk)

🎯 Affected (1)
  • Domain-wide risk
🛠 Remediation

1. Configure lockout (threshold:5, duration:15min)
2. Increase minimum password length to 14+
3. Enable password complexity
