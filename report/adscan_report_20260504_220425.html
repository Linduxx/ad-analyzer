AD-005krbtgt Password Never Changed — Golden Ticket Risk
🔴 CRITICALKerberos▼
📋 Description

The krbtgt password has NEVER been changed.
Attacker with this hash can forge Golden Tickets for permanent domain access.

⚡ Exploitable By
⚡ Domain Admin
📍 Found In

LDAP → pwdLastSet attribute on krbtgt account (value: 0 / null)

🎯 Affected (1)
  • krbtgt
🛠 Remediation

Reset krbtgt password TWICE (10 hours apart).
Use Microsoft Reset-KrbtgtPassword script. Rotate every 180 days.

AD-001AS-REP Roasting — Pre-authentication Disabled
🟠 HIGHKerberos▼
📋 Description

1 account(s) have Kerberos pre-authentication disabled.
Attacker can request AS-REP ticket without credentials and crack offline.

⚡ Exploitable By
⚡ Low-privileged domain user
📍 Found In

LDAP → userAccountControl (flag: DONT_REQ_PREAUTH = 0x400000)

🎯 Affected (1)
  • test.user
🛠 Remediation

Set-ADAccountControl -Identity -DoesNotRequirePreAuth $false
ADUC → Account tab → uncheck 'Do not require Kerberos preauthentication'
